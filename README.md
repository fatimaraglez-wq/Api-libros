# API Libros - Laravel 13

## Descripción

API REST desarrollada con Laravel 13 para la gestión de libros.

## Repositorio

https://github.com/fatimaraglez-wq/Api-libros

## Requisitos

* Debian 12
* Conexión a Internet
* Permisos de administrador (sudo)

## Instalación

Clonar el repositorio:

git clone https://github.com/fatimaraglez-wq/Api-libros.git

Entrar al proyecto:

cd Api-libros

Dar permisos al script:

chmod +x instalar_api.sh

Ejecutar la instalación:

sudo ./instalar_api.sh

## Base de Datos

El script crea automáticamente la base de datos:

api1

y ejecuta las migraciones necesarias.

## Acceso

Una vez finalizada la instalación, la API estará disponible en:

http://IP_DEL_SERVIDOR

Ejemplo:

http://IP_DEL_SERVIDOR/api/libros
